# Awesome QA

A react app to manage a knowledge base system. Has the ability to create, read, update and delete questions in the knowledge base

## Setup

This project was bootstrapped with [https://github.com/facebook/create-react-app](Create React App). Clone the project on github and run the `npm install` or `yarn` command

## Available Scripts

In the project directory, you can run:

### `npm start` or `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm test` or `yarn test`

Launches the test runner in the interactive watch mode.
